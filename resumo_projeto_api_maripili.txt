
RESUMO DO PROJETO: API RESTAURANTE MARIPILI

🎯 Objetivo final
Criar uma API pública em FastAPI que consulte um banco de dados MySQL online, retornando informações reais do restaurante Maripili (como pratos ativos, preços, categorias, etc).

🧱 Estrutura do Projeto

1. Código da API
- Linguagem: Python
- Framework: FastAPI
- Arquivo principal: busca_pratos_api.py

2. Banco de Dados
- MySQL remoto (freemysqlhosting.net)
- Contém tabela com pratos, colunas como:
  Nome, Restaurante, classificação gastrono, ativo/suspenso, Preço_venda, etc.

3. Hospedagem da API
- Plataforma: Render.com
- Tipo de serviço: Web Service (Free)
- Deploy automatizado via GitHub

🔗 Conexões necessárias

1. API precisa se conectar ao MySQL
   → Usando mysql.connector e variáveis de ambiente (DB_HOST, DB_PORT, DB_USER, DB_PASS, DB_NAME)

2. Render precisa puxar o código do GitHub automaticamente
   → Isso exige que o projeto local esteja versionado com Git e conectado a um repositório remoto (no GitHub)
